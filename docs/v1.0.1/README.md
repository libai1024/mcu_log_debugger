## 🎯 v1.0.1 核心功能

### 1. 三种显示模式 🔄

| 模式 | 说明 | 使用场景 |
|------|------|---------|
| **Log** | 结构化日志显示 | 日常开发调试 |
| **HEX** | 十六进制 + ASCII | 协议分析、二进制数据 |
| **Normal** | 纯文本终端 | 原始数据流查看 |

**快捷键**: `Cmd/Ctrl + 1/2/3`

### 2. 视图锁定 🔒 ⭐

**功能**: 在搜索或查看历史时，冻结视图，防止新消息干扰

**触发方式**:
- 🔍 搜索时自动锁定
- 🛑 关闭自动滚动时锁定
- 📜 手动滚动到非底部时锁定
- 🔒 点击锁定按钮手动锁定

**锁定时**:
- 视图保持不变
- 新消息后台累积
- 右下角显示新消息数量
- 可随时解锁查看

**快捷键**: `Cmd/Ctrl + L`

### 3. 设置入口优化 ⚙️

**改进**: 移至右上角图标

**位置**:
```
┌─────────────────────────────────────────────────┐
│  MCU Log Debugger           🌙  ⚙️  ❓  ✕      │
└─────────────────────────────────────────────────┘
```

**快捷键**: `Cmd/Ctrl + ,`

## 📝 开发规范

### 代码风格
- JavaScript: ES6+, 使用 `const`/`let`
- CSS: BEM 命名规范
- Rust: 遵循 Rustfmt

### 注释规范
- 函数必须有 JSDoc 注释
- 复杂逻辑必须有行内注释
- 所有 Tauri 命令必须有文档注释

### 提交规范
- 使用 Conventional Commits
- 提交信息清晰明了
- 每个提交只做一件事
