# v1.0.1 Release Notes

## 概述
v1.0.1 版本重点提升了日志的展示灵活性与交互稳定性，引入了核心的“显示模式切换”与“视图锁定”机制。

---

## 新增功能

### 1. 三大显示模式 🔄
支持根据不同调试需求切换显示风格：
- **Log 模式**：结构化表格显示，适合日常日志分析。
- **HEX 模式**：十六进制 + ASCII 对照，适合二进制协议调试。
- **Normal 模式**：原始文本流显示，模拟传统终端体验。
- **快捷键**：`Cmd/Ctrl + 1/2/3` 快速切换。

### 2. 视图锁定 (View Lock) 🔒
解决高频日志滚动时难以查看历史的问题：
- **自动锁定**：搜索关键字或向上滚动查看历史时，视图自动冻结。
- **后台累积**：锁定时新消息在后台缓冲，并实时显示未读数。
- **一键解锁**：点击解锁按钮或跳至新消息即可恢复实时滚动。
- **快捷键**：`Cmd/Ctrl + L`。

### 3. 设置入口优化 ⚙️
- 设置按钮移至右上角标题栏，保持侧边栏清爽。
- 优化了弹窗的唤起动画与层级。
