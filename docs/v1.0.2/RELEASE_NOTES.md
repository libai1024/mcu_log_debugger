# MCU Log Debugger v1.0.2 发布说明

## 🎉 版本亮点

v1.0.2 是一个优化版本，专注于提升用户体验和功能增强。

### 核心改进
- 🐛 修复串口图标显示问题
- 🎨 简化界面，移除冗余提示
- ⭐ 级别筛选改为多选模式
- 🔍 搜索功能大幅增强
- 🎨 搜索 UI 全面优化

---

## ✨ 新功能

### 1. 级别多选筛选 ⭐
**旧方式**：下拉框单选（全部/DEBUG及以上/INFO及以上...）  
**新方式**：复选框多选（VERBOSE/DEBUG/INFO/WARN/ERROR）

**优势**：
- ✅ 更灵活：可以只看 ERROR + WARN
- ✅ 更直观：一眼看到选中的级别
- ✅ 更快速：无需打开下拉框

**示例**：
```
只看错误和警告：
☐ VERBOSE
☐ DEBUG
☐ INFO
☑ WARN
☑ ERROR
```

### 2. 正则搜索增强 🔍
- ✅ **全词匹配**：新增 "ab|" 按钮
  - 搜索 "error" 只匹配完整单词，不匹配 "errors"
- ✅ **正则快捷按钮**：5 个常用正则表达式一键填充
  - `.*error.*` - 包含 error
  - `^[ERROR]` - ERROR 开头
  - `\d{2}:\d{2}:\d{2}` - 时间格式
  - `0x[0-9A-Fa-f]+` - 十六进制
  - `\b\d+\b` - 纯数字
- ✅ **错误提示**：正则语法错误时显示友好提示
- ✅ **图标优化**：.* / Aa / ab| 更专业简洁

### 3. 搜索 UI 优化 🎨
- ✅ 布局重新设计：选项和计数器同行
- ✅ 导航按钮独立一行：更大更易点击
- ✅ 正则快捷按钮：折叠显示，不占空间
- ✅ 计数器样式：背景色区分，等宽字体

---

## 🐛 Bug 修复

### 串口图标显示问题
- **问题**：串口标签旁边的 "?" 提示图标可能导致布局问题
- **修复**：移除所有 form-hint 提示图标
- **影响**：界面更简洁，图标显示完整

### 代码优化

- 错误处理和降级策略
